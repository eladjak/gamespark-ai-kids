# GameSpark AI

אתר ווב אינטראקטיבי בעברית וב-RTL עבור פלטפורמה שבה ילדים יוצרים משחקים בעזרת בינה מלאכותית.

## מה יש בפרויקט

- סטודיו יצירה פעיל עם רעיון למשחק, בחירת ז'אנר, דמות ועולם.
- Canvas preview שמתעדכן לפי הבחירות של המשתמש.
- מצב טעינה מדומה ליצירת פרוטוטייפ.
- גלריית משחקים לדוגמה.
- אזור אמון להורים ולמורים.
- עיצוב רספונסיבי לדסקטופ ומובייל.

## הרצה מקומית

```bash
python -m http.server 4178
```

לאחר מכן לפתוח:

```text
http://127.0.0.1:4178
```

## קבצים חשובים

- `index.html` - מבנה האתר והתוכן בעברית.
- `styles.css` - שפת העיצוב, רספונסיביות ואנימציות.
- `script.js` - אינטראקציות הסטודיו וציור ה-Canvas.
- `gamespark-ai-home.png` - תמונת קונספט/צילום דסקטופ.
- `gamespark-ai-mobile.png` - צילום בדיקת מובייל.
